# 🎓 Student Marks Prediction using Machine Learning

## 📌 Project Overview
This project predicts student marks based on study hours using Linear Regression.

## 🚀 Features
- Data Visualization (Seaborn & Matplotlib)
- Regression Model
- Prediction System
- User Input Support

## 📊 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📁 Dataset
The dataset contains:
- Study Hours
- Student Marks
